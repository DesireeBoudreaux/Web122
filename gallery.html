This website was created as a student project for Chapters 10–17 to demonstrate mobile‑first design, semantic HTML structure, accessible color practices, and modern CSS techniques. Earlier chapters focused on layout, spacing, typography, and responsive navigation, while later chapters introduced visual enhancements such as gradient‑styled buttons, soft shadows with hover transitions, blend‑mode effects on imagery, and a floated portrait image with rounded shaping for natural text wrapping. Chapters 15–17 added interactive motion through a smooth card hover transition, a subtle transform effect, and a soft fade‑and‑rise animation applied to section titles. Theme colors were defined in the CSS :root using HEX fallbacks and an OKLCH accent color, and all color combinations were checked for readability.
