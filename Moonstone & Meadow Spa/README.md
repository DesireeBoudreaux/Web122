Moonstone & Meadow Spa
A mobile‑first, 5‑page fictional spa website built for Web122 – CSS Techniques & Projects.
This project demonstrates responsive layout, semantic HTML, modern CSS effects, and accessible design practices.

Overview:
Moonstone & Meadow Spa is a fictional, spa‑themed website created solely for educational purposes.
It presents a calm, nature‑inspired brand identity designed to showcase pretend spa services, gallery images, and business information. The site emphasizes soft color palettes, gentle motion, and clean typography to reflect a peaceful aesthetic.Features

Features:
Fully responsive, mobile‑first layout
Sticky header with hamburger navigation
3‑column responsive grid using CSS Grid
CSS transitions, transforms, and animations
Container queries for adaptive card layouts
Floated image with shape-outside for natural text wrapping
Gradient buttons, soft shadows, and blend‑mode image effects
Semantic HTML structure for accessibility and readability

Technologies Used:
HTML5 • CSS3 • JavaScript • Google Fonts
